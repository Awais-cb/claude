## How to Use These Guides

Each guide is structured as:
1. **Setup** — one-time configuration for that workflow
2. **Daily workflow** — the prompts and commands used day-to-day
3. **Prompt library** — copy-paste prompts for common tasks
4. **Tips** — how to get the most out of Claude for that role

These are living documents. As your team discovers better prompts or patterns, update them and commit — everyone benefits.
