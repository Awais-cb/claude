# Laravel Workflows — Claude Code

Role-specific workflow guides for Laravel projects. Whether you're building APIs, crafting Blade/Livewire UIs, or owning the full stack, these guides show how to use Claude effectively at every step.

---

## Workflow Guides

| Guide | Who it's for | What it covers |
|-------|-------------|----------------|
| [fullstack.md](fullstack.md) | Fullstack Devs | End-to-end feature: migration → service → controller → Blade/Livewire → tests |
| [backend.md](backend.md) | Backend Devs | APIs, service layer, jobs, queues, auth, artisan commands |
| [frontend.md](frontend.md) | Frontend Devs | Blade, Livewire, Alpine.js, Inertia.js, Vite, Tailwind |
| [database.md](database.md) | All Devs | Migrations, Eloquent models, relationships, seeders, query optimization |
| [testing.md](testing.md) | All Devs | Pest/PHPUnit, feature tests, unit tests, mocking, factories, Dusk |
| [pr-review.md](pr-review.md) | All Devs | Reviewing Laravel PRs via Claude Code — security, N+1s, migrations, logic |

---

## How to Use These Guides

Each guide follows the same structure:
1. **Session setup** — how to start a productive Claude session for this role
2. **Phase-by-phase workflows** — prompts for each stage of development
3. **Prompt library** — copy-paste prompts for common tasks
4. **Tips** — Laravel-specific advice for getting the best results
